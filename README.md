# tsukuba-weather-ical
本リポジトリは、気象庁の最新予報データ（JSON）を取得し、Googleカレンダー等に登録可能なiCalendar（.ics）形式に自動変換・更新するシステムです。毎日決まった時間につくば市の詳細な天気をカレンダーへ自動反映します。

## コンセプト
* 既存のiCalendar形式の天気予報提供サービスでは「つくば」の地点情報が不足していました。
* 気象庁の週間予報データは広域となっていますが、本ツールでは「当日・明日・明後日」に特化した**詳細な予報データ**を取得することで、より正確な予定管理を可能にしました。
* Googleカレンダーの通知機能と組み合わせることで、毎朝アプリを開く手間を省き、自動的にその日の天気を把握できる環境を構築しました。

## 主な機能
* 気象庁が提供する最新の予報JSONデータを自動取得。
* `icalendar`ライブラリを使用し、終日イベントとして最高/最低気温、降水確率を含むイベントを生成。
* GitHub Actionsを利用し、毎日日本時間午前6時にデータを自動更新。

## 技術スタック
* **言語**: Python 3.x
* **主なライブラリ**: 
    * `requests`（API通信）
    * `icalendar`（iCal形式生成）
* **インフラ・自動化**: GitHub Actions
* **データソース**: 気象庁 予報JSON（茨城県南部・土浦地点）

## 動作の仕組み
1.  **データ取得**: 毎朝6:00(JST)にGitHub Actionsが起動し、気象庁から茨城県南部の予報データを取得します。
2.  **パース・処理**: 取得したJSONから「天気」「降水確率」「最高/最低気温」を抽出します。
3.  **ファイル生成**: `tsukuba_weather.ics` ファイルを作成・更新します。
4.  **デプロイ**: 更新されたファイルをGitHubリポジトリへ自動コミット＆プッシュし、最新のURLをカレンダー側から参照（外部カレンダーとして登録）可能な状態にします。

## 使用方法
1.  本リポジトリで生成された `https://t087t.github.io/tsukuba-weather-ical/tsukuba_weather.ics` のURLをコピーします。
2.  Googleカレンダーを開き、「他のカレンダー」の＋アイコンから「URLで追加」を選択します。
3.  コピーしたURLを貼り付けることで、カレンダー上につくばの天気が表示されます。
4.  設定から「予定の通知」を有効にすることで、毎朝天気の通知を受け取ることが可能です。
